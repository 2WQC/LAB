## Overview

A specific configuration of stocks is being assembled in order to create a one-time dataset. This dataset will serve as a foundation for testing and understanding different analytical concepts.

## Background

The idea is based on the following principle:

A well-known and clearly defined dataset can be fully analyzed and deeply understood. This makes it easier to interpret and apply new analytical approaches to future datasets.

## Procedure

- A dataset is created once and used as the initial basis for implementing analytical methods.
- In parallel, the selected stocks are continuously monitored in real time.
- Relevant events are recorded as they occur.
- After a defined period, the dataset is rebuilt and expanded to include these newly captured events.
- This process is intended to run continuously over multiple years.
